# Michael-Wang-APS360-Project-Repository
A repository of the coronary artery disease severity classification  project. The project follows the following architecture: (0) Results folder consisting of all training curves (1) Labeling and data processing and data histogram (2) Baseline model and baseline training (3) Resnet transfer learning and training (4) YOLOv8 ROI identification (5) MedViT classification of stenosis severity NOTE: Please visit https://github.com/Omid-Nejati/MedViT/blob/main/README.md for the MedViT implementation.
